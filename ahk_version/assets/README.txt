============================================================
  TimestampTool 图标资源说明
============================================================

请将应用程序图标文件放置在此目录：

  assets/icon.ico

要求：
- 格式：ICO 文件（Windows 图标格式）
- 建议尺寸：包含 16x16、32x32、48x48、256x256 多种尺寸
- 用途：编译后的 EXE 文件图标和系统托盘图标

如果没有自定义图标，编译时可去掉 /icon 参数，
将使用 AutoHotkey 默认图标。

推荐图标制作工具：
- https://www.favicon-generator.org/
- https://icoconvert.com/
- GIMP（免费图像编辑器，支持导出 ICO）
